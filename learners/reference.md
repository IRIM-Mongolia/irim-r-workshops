Семинарт ашигласан функцүүдийн мэхлэх хуудас. Энэ нь биднийг семинараар дамжуулан ажиллахад хүн амтай байх болно.

- `install.packages()` # `R` багцуудыг компьютер дээрээ суулгана уу

## Семинар -- R багцуудын танилцуулга`, `Markdown` and `Notebooks`
- `library()` # `R` сесс бүрт `R` багцыг ачаалах

## Семинар -- Өгөгдлөөс эхэлнэ
- `<-` # оноох сум нь нэрлэсэн объектод оролт оноодог
- `read_csv()` # .`csv` файл дахь өгөгдлийг `R` руу импортлох (`readr`, `tidyverse`-ийн хэсэг)
- `class()` # объектын ангиллыг шалгана уу
- `dim()` # нь объектын хэмжээс бүхий векторыг буцаана
- `nrow()` # нь мөрийн тоог буцаана
- `ncol()` # нь баганын тоог буцаана
- `head()` # эхний 6 мөрийг харуулж байна
- `tail()` # нь сүүлийн 6 мөрийг харуулж байна
- `names()` # нь баганын нэрийг буцаана
- `str()` # объектын бүтэц, багана бүрийн ангилал, урт, агуулгын талаарх мэдээлэл
- `summary()` # багана тус бүрийн хураангуй статистик
- `glimpse()` # нь tibble-ийн багана, мөрийн тоо, багана тус бүрийн нэр, ангиллыг буцаадаг бөгөөд дэлгэцэн дээрх олон утгыг урьдчилан харах боломжтой (`dplyr`, `tidyverse`-ын хэсэг)
- `c()` # утгыг вектор эсвэл жагсаалт болгон нэгтгэнэ
- `factor()` # векторыг хүчин зүйл болгон кодлох
- `levels(0` # нь хувьсагчийн түвшний шинж чанарт хандах боломжийг олгодог
- `nlevels()` # хүчин зүйлийн түвшний тоог өгдөг
- `as.character()` # тэмдэгтийн төрлийн объект үүсгэх эсвэл албадах
- `as.numeric()` # тоон төрлийн объект үүсгэх эсвэл албадах
- `as.factor()` # векторыг хүчин зүйл болгон кодлох
- `plot()` # ерөнхий X-Y график үүсгэнэ
- `day()` # огнооноос өдрийн утгыг гаргаж авах (`lubridate`, `tidyverse`-ын хэсэг)
- `month()` # огнооноос сарын утгыг гаргаж авах (`lubridate`, `tidyverse`-ын хэсэг)
- `year()` # огнооноос жилийн утгыг гаргаж авах (`lubridate`, `tidyverse`-ын хэсэг)
- `day()` # огнооноос өдрийн утгыг гаргаж авах (`lubridate`, `tidyverse`-ын хэсэг)
- `as_date()` # объектыг огноо эсвэл огноо руу хөрвүүлэх (`lubridate`, `tidyverse`-ын хэсэг)
- `mdy()`, `ymd()`, `dmy()` # огноог задлах (`lubridate`, `tidyverse`-ийн хэсэг)

## Семинар -- `dplyr` болон `tidyr`-тэй өгөгдлийг удирдах
- `select()` # дэд багана (`dplyr`, `tidyverse`-ын хэсэг)
- `filter()` # дэд багц мөр нөхцөлүүд (`dplyr`, `tidyverse`-ын хэсэг)
- `mutate()` # бусад баганын мэдээллийг ашиглан шинэ багана үүсгэх (`dplyr`, `tidyverse`-ын хэсэг)
- `group_by()` болон `summarize()` # бүлэглэсэн өгөгдлийн хураангуй статистикийг үүсгэдэг (`dplyr`, `tidyverse`-ын хэсэг)
- `arrange()` # илэрцийг эрэмбэлэх (`dplyr`, `tidyverse`-н хэсэг)
- `count()` # салангид утгыг тоолох (`dplyr`, `tidyverse`-ын хэсэг)
- ` %>% ` # хоолой, нэг функцын гаралтыг аваад дараагийнх руу шууд илгээнэ үү (`tidyverse`)
- `filter()` # шүүлтүүр мөр (`dplyr`, `tidyverse`-н хэсэг)
- `select()` # багана сонгох (`dplyr`, `tidyverse`-ын хэсэг)
- `mutate()` # багана үүсгэх, өөрчлөх, устгах (`dplyr`, `tidyverse`-ын хэсэг)
- `distinct()` # ялгаатай/өвөрмөц мөрүүдийг хадгалах (`dplyr`, `tidyverse`-ын хэсэг)
- `sample_n()` # хүснэгтээс n мөр түүвэрлэх (`dplyr`, `tidyverse`-ын хэсэг)
- `pivot_wider()` # дата уртаас өргөн рүү эргэх (`tidyr`, `tidyverse`-ын хэсэг)
- `pivot_longer()` # пивот өгөгдөл өргөнөөс урт руу (`tidyr`, `tidyverse`-ын хэсэг)
- `separate_longer_delim()` # мөрийг мөр болгон хуваах (`tidyr`, `tidyverse`-ын хэсэг)
- `replace_na()` # NA-г заасан утгуудаар солих (`tidyr`, `tidyverse`-ын хэсэг)
- `write_csv()` # өгөгдлийг .`csv` форматаар экспортлох (`readr`, `tidyverse`-ын хэсэг)
